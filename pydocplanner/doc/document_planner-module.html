bottom_up_plan(messages, rules, dtype=None, text=None)

source code 

The main method implementing the Bottom-Up document structuring algorithm from "Building Natural Language Generation Systems" figure 4.17, pg 108.

The method takes a list of Messages and a set of Rules and creates a document plan by repeatedly applying the highest-scoring Rule-application (according to the Rule's heuristic score) until a full tree is created. This is returned as a DocPlan with the tree set as children.

If no plan is reached using bottom-up, None is returned.

Parameters:
  • dtype (string)
  • messages (list of Message)
  • rule (list of Rule)
  • text (string)
